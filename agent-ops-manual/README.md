# Studio‑Grade PDF (Node + Tailwind + Playwright)

This is a starter pack for generating premium PDFs using **HTML + Tailwind + Playwright**.

## Quickstart

```bash
npm i
npm run pdf
```

It will render `templates/report.html` to `report.pdf` (in the project root).

## Swap to Puppeteer (optional)

- Install puppeteer and replace renderer logic if you prefer. Playwright tends to be more stable across environments.
