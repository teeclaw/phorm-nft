# Life Knowledge Graph (PARA)

This is Layer 1 durable memory.

## Structure
- projects/  active initiatives with outcomes, status, next actions
- areas/     ongoing responsibilities (health, finance, ops, relationships, etc.)
- resources/ reference material and reusable knowledge
- archives/  inactive/finished content
- people/    durable facts about people, preferences, context

## Rules
1. Keep entries concise and factual.
2. Add/update summary files for quick lookup.
3. Move stale content to archives.
4. Prefer one canonical file per project/person, then link out.
