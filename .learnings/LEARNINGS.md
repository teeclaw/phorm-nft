## 2026-03-04 - CEO Must Check Department Logs Before Task Assignment
**What happened:** Owner requested A2A endpoint upgrade. TeeClaw gave a full assignment plan and timeline, but TeeCode had already completed the work 1.5 hours earlier. The daily log had the answer.
**Root cause:** Didn't read today's department logs before responding to a task request.
**Fix:** ALWAYS read today's `memory/YYYY-MM-DD-*.md` files before accepting or delegating any task. No exceptions.
**Promote to:** AGENTS.md (CEO operating procedure)

## 2026-03-04 - X/Twitter Pricing Model Change
**What happened:** Owner corrected outdated knowledge. X (Twitter) no longer uses subscription tiers (Free/Basic/Premium). They now use pay-per-usage pricing.
**Impact:** All references to "Premium tier", "Free tier", "Basic tier" for X accounts are outdated. Account capability is now usage-based, not tier-based.
**Fix:** Stop referencing X subscription tiers. Use "pay-per-usage" model when discussing X account capabilities and costs.
**Promote to:** TOOLS.md (X account notes), MEMORY.md
